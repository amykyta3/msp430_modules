Detailed Description

Initializes the clock system.

Author
Alex Mykyta

This module provides a generic method of initializing an MSP430's system clocks. Using the configuration file, the clock sources for MCLK SMCLK and ACLK are defined as well as any clock division. If a clock network uses a DCO/PLL as its source, the clock system configures that as well to produce the desired frequency.
